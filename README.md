# 🔥 Hong Kong Fire Timeline | 香港火灾时间线

一个追踪和展示香港火灾事件的时间线网页项目。通过数据可视化方式记录重要的火灾事件和安全信息。

## 📱 项目特点

- 📊 **时间线展示** - 清晰的时间线界面展现各个火灾事件
- 🎯 **事件追踪** - 记录火灾发生时间、地点和相关信息
- 💻 **响应式设计** - 支持桌面和移动设备浏览
- 🌐 **在线访问** - GitHub Pages 自动部署

## 🚀 快速开始

### 在线查看
访问项目主页: [https://xgyzbb.github.io/default/](https://xgyzbb.github.io/default/)

### 本地运行
1. Clone 仓库：
```bash
git clone https://github.com/xgyzbb/default.git
cd default
```

2. 打开 `index.html` 文件
```bash
# 方式一：直接在浏览器中打开
open index.html

# 方式二：使用本地服务器（推荐）
python -m http.server 8000
# 然后访问 http://localhost:8000
```
